Anki → Slides

Drop an Anki plain-text export and download a 16:9 slide deck. Front and back become separate slides, in order. Nothing is stored — files are converted on the fly.

Which Anki export should I use?

Only one of Anki's export options is supported. Pick Notes in Plain Text (.txt).

Anki export optionSupported?Why
Notes in Plain Text (.txt) Yes — use this Tab-separated front/back; image references (<img src>) survive when “Include HTML” is on.
Cards in Plain Text (.txt) Works Same tab-separated shape, one row per card instead of per note.
Anki Deck Package (.apkg) No Binary SQLite bundle. Not parsed — convert to Notes in Plain Text first.
Anki Collection Package (.colpkg) No Full profile backup; same binary format as .apkg.
PDF / HTML exporter (add-on) No Those are output formats, not inputs.
Exact export steps (with images)
  1. In Anki desktop: File → Export…
  2. Export format: Notes in Plain Text (.txt)
  3. Include: pick one deck, or leave All Decks.
  4. Check Include HTML and media references.
    Without it, Anki strips the <img> tags and your images can't be matched — you'll get a text-only deck even if you zip media.
  5. Leave “Include tags” and “Include deck name” off (they'd clutter the slides).
  6. Save the .txt.
  7. Find your media folder:
    • macOS: ~/Library/Application Support/Anki2/<profile>/collection.media
    • Windows: %APPDATA%\Anki2\<profile>\collection.media
    • Linux: ~/.local/share/Anki2/<profile>/collection.media
    Or from Anki: Tools → Check Media… shows the path.
  8. Select the .txt and the collection.media folder in your file manager, right-click → Compress (macOS) or Send to → Compressed folder (Windows). Drop the resulting .zip onto the upload box.

Nested folders inside the zip are fine. Images are matched by filename, so collection.media/abc.png and abc.png both resolve.

What makes it into the slide vs. what gets dropped
ContentResult
Card front / back textOne slide each (front1, back1, front2, back2, …)
Line breaks (<br>, newlines)Preserved
Images (<img src>)Embedded — but only if you upload the .zip with media
Bold / italic / color / fontsStripped — slides use a single clean typeface
MathJax / LaTeX (\[ … \])Kept as raw source, not rendered
Audio / video ([sound:…] tags)Silently dropped — slide decks can't play them
Cloze deletions ({{c1::…}})Shown as raw text; export with “Include HTML” to see the card's actual rendered HTML.
Tags, deck name, scheduling dataNot included
Runs entirely on a serverless function · no uploads are persisted
