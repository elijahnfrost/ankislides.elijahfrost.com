Anki → Slides

Drop any Anki export and download a 16:9 slide deck. Front and back become separate slides, in order. Nothing is stored — files are converted on the fly.

Which Anki export should I use?

Easiest is Anki Deck Package (.apkg) — single file, images bundled. All four input types below are supported; the right-side table summarizes what survives.

Anki export optionSupportedNotes
Anki Deck Package (.apkg) Yes — recommended One file, images bundled, nothing to tick. Works for the newest zstd-compressed bundles too.
Anki Collection Package (.colpkg) Yes Full-profile backup. Every note in the collection becomes slides — so it can be big.
Notes in Plain Text (.txt) Yes Text only. To include images, tick Include HTML and media references and upload as a zip with collection.media.
Cards in Plain Text (.txt) Yes Same shape as Notes, one row per card rather than per note.
PDF / HTML exporter (add-on) Output only Those are result formats, not inputs.
Using a .txt export with images
  1. In Anki: File → Export… → Notes in Plain Text (.txt).
  2. Tick Include HTML and media references. Without this the <img> tags never make it into the file, so the images can't be matched later.
  3. Leave “Include tags” and “Include deck name” off — they clutter the slides.
  4. Find your media folder — Tools → Check Media… shows the path. Defaults:
    • macOS: ~/Library/Application Support/Anki2/<profile>/collection.media
    • Windows: %APPDATA%\Anki2\<profile>\collection.media
    • Linux: ~/.local/share/Anki2/<profile>/collection.media
  5. Select the .txt and the collection.media folder, right-click → Compress. Drop the resulting .zip.

Or skip all of this and export an .apkg instead — it already contains everything.
